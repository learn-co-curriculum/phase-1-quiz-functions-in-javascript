# Functions: Hoisting

For hoisting to work, the function must be defined as a function declaration.

For example:

```javascript
// Hoisting
willItHoist();

// Function declaration
function willItHoist() {
  console.log("You tell me!");
}
```

## Correct

True

> Correct!

## Incorrect

False
