# Functions: Syntax

Select all the ways you can write a function.

## Correct

```javascript
function foo() {
  return "bar";
}
```

## Correct

```javascript
let foo = function () {
  return "bar";
};
```

## Correct

```javascript
const foo = () => "bar";
```

## Correct

```javascript
const foo = () => {
  return "bar";
};
```

## Incorrect

```javascript
const foo => { return 'bar'; }
```
